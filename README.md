# 📱 Mood Tracker App

A modern iOS application built with **SwiftUI** that helps users track their daily emotions, identify mood patterns, and develop healthy self-reflection habits. Users can quickly log their mood, add optional notes, view previous entries, analyze mood trends through charts, and receive daily reminders to check in. 

## 👥 Contributors

This project was developed by **Heer Soni** in collaboration with **Carley Rodenbush** as part of an iOS application development course. Heer led the implementation and development of the application while collaborating with Carley on the overall project design and functionality.

## ✨ Features

- 😊 Log daily moods with emoji-based mood selection
- 📝 Add optional notes to each mood entry
- 📖 View recent mood history
- 📊 Visualize mood trends with interactive charts
- 📈 View average mood score and statistics
- 💡 Receive personalized mood insights and recommendations
- 🔔 Enable daily mood reminder notifications
- 💾 Store mood entries locally using SwiftData
- 🎨 Modern and intuitive SwiftUI interface

## 📱 App Workflow

1. Launch the app.
2. Select your current mood.
3. Optionally add a note.
4. Save the mood entry.
5. Review previous entries in the **History** tab.
6. Explore mood trends and insights in the **Charts** tab.
7. Receive optional daily reminders to continue tracking your mood.

## 🔔 Notifications

The app includes optional daily reminders to encourage consistent mood tracking. Users can enable notifications directly within the app and manage notification preferences through the iOS Settings app.

## 🎯 Project Purpose

Mood Tracker was designed to make emotional self-reflection simple and accessible. By allowing users to quickly record how they feel and visualize patterns over time, the app encourages greater awareness of emotional well-being and supports healthy daily habits.
